### 📁 Projects

#### 🔐 [iron_vault](https://github.com/cpericaro/iron_vault)
> CLI password manager written in Rust

- **AES-256-GCM** encryption
- **Argon2id** key derivation
- Zero-Knowledge architecture — your master password never leaves your machine
- Passwords are physically zeroed in RAM on scope exit using the [`zeroize`](https://crates.io/crates/zeroize) crate
